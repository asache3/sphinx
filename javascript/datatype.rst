==========
データ型
==========

配列リテラル(array)
=====================

::

  var ary = ['JavaScript', 'AJax', 'ASP.NET'];
  document.writeln(ary[0]);


オブジェクトリテラル(object)
==============================

* 名前をキーにアクセスできる配列
* ハッシュ、連想配列

::

  var obj = {x:1, y:2, z:3};
  document.writeln(obj.x);
  document.writeln(obj['x']);
