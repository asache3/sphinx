========
Object
========

変数
======

* ローカル変数

  * 先頭がアルファベットの小文字か「_」で始まる

* グローバル変数

  * 先頭が「$」で始まる

* インスタンス変数

  * 先頭が「@」で始まる

* クラス変数

  * 先頭が「@@」で始まる


定数
======

* アルファベット大文字で始まる
* 一度定義された定数に再び代入を行おうとすると警告メッセージが出る
* 外部から参照するためには"::"を用いる


オブジェクトが同じかどうかを調べる
====================================

2つのオブジェクトが同じかどうか(IDが同じかどうか)を調べる
-------------------------------------------------------------

::

  p str1.equal?(str2)


オブジェクトの値が等しいかどうかを調べる
------------------------------------------

::

  p 1.0 == 1 #=> true

  p 1.0.eql? 1 #=> false
  # 厳密に比較を行う
  
  p((1..3) === 2) #=> true
  p(/zz/ === "xyzzy") #=> true
  p(String === "xyzzy") #=> true
  # 緩い意味で同じかどうかを判断する
