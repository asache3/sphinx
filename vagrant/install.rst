=========
Install
=========

Install VirtualBox
====================

`Download VirtualBox <https://www.virtualbox.org/wiki/Downloads>`__
から、環境に合ったものをダウンロードしてインストール


Install Vagrant
=================

`Vagrant Downloads <http://downloads.vagrantup.com>`__
から、環境に合ったものをダウンロードしてインストール

または、rubygemsに登録されているので、

::

  gem install vagrant
