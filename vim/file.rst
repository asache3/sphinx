===============
Multiple File
===============

バッファを一覧表示する
==========================

::

  :ls


バッファを切り替える
======================

::

  :bnext # 次のバッファ
  :bprev # 前のバッファ
  :bfirst # 先頭のバッファ
  :blast # 末尾のバッファ
  :buffer N # 番号を指定する


バッファを削除する
====================

::

  :bdelete N1 N2 N3
  :N,M bdelete

  :bd 5 6 7 9 10
  :5,10bd


ウィンドウを分割する
======================

.. csv-table::
  :header-rows: 1

  動作,コマンド
  ウィンドウを水平分割し、現在のバッファの内容を利用する,<Ctrl-w>s
  ウィンドウを垂直分割し、現在のバッファの内容を利用する,<Ctro-w>v
  ウィンドウを水平分割し、{file}を読み込む,:sp[lit] {file}
  ウィンドウを垂直分割し、{file}を読み込む,:vsp[lit] {file}

==================================

.. csv-table::
  :header-rows: 1

  上のウィンドウにフォーカス,<Ctrl-w>k
  右のウィンドウにフォーカス,<Ctrl-w>l

==================================

.. csv-table::
  :header-rows: 1

  動作,コマンド
