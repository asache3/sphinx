=======
Tuple
=======

* タプルがリストと異なるのは、要素を書き換えることができないという点
* タプルはディクショナリのキーとして利用できる

タプルを定義する
=================

::

  t = (1, 2, 3, 4, 5)
  ipmap = {(0, 1, 2, 3):"host.name.com",
           (0, 2, 1, 3):"some.dname.com"}


リストに変換する
==================

::

  t = (1, 2, 3, 4, 5)
  a = list(t)


タプルに変換する
==================

::

  t2 = tuple(a)
