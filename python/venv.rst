======
venv
======

* Pythonの仮想環境を整えるモジュール
* Python3.3から標準になった

仮想環境を作る
================

::

  # pyvenvコマンドを実行
  pyvenv [仮想環境ディレクトリ]

  または

  # venvモジュールを実行
  python -m venv [仮想環境ディレクトリ]


仮想環境をアクティブにする
============================

::

  # Linuxなどの場合
  source bin/activate

  # Windowsの場合
  bin/activate
