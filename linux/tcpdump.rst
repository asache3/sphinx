=========
tcpdump
=========

実行する
==========

::

  tcpdump


フィルタをかける
==================

ポートで絞る
--------------

::

  tcpdump port 80


送信/受信だけに絞る
---------------------

::

  tcpdump dst port 80 # 送信
  tcpdump src port 80 or src port 443 # 受信


ホストで絞る
--------------

::

  tcpdump host 192.168.1.1
  tcpdump dst host 192.168.1.1
  tcpdump src host 192.168.1.1


ネットワークアドレスで絞る
----------------------------

::

  tcpdump net 192.168.2.0 mask 255.255.255.0


ファイルに保存する
====================

::

  tcpdump -w [ファイル名]
