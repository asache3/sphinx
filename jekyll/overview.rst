==========
Overview
==========

Jekyllとは
============

* 静的サイトジェネレータ
* マークダウン記法を使える


Install
=========

::

  gem install jekyll


ひな型を作る
==============

::

  jekyll new sample


Webサーバーを起動する
=======================

::

  jekyll serve
  jekyll serve --watch # ファイルの変更を監視する


HTMLファイルに変換する
========================

::

  jekyll build
